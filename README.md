## Restaurant Website

This is a basic restaurant website built using HTML and CSS. 
It is a single-page application that displays the restaurant menu at the top 
and a contact form at the bottom. The website is designed with a clean layout 
to provide easy access to menu details and customer contact information.

### Features
- Single-page layout
- Menu section displaying food items
- Contact form for customer inquiries
- Simple and user-friendly design
- Responsive layout using basic CSS

### Technologies Used
- HTML5
- CSS3

### Purpose
This project was created to practice basic front-end web development skills 
and understand webpage layout and form design.
